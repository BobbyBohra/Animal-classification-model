# 🐾 Animal Classification Web App

A deep learning based Animal Classification Web Application built using TensorFlow, Flask, and MobileNetV2.  
This project can classify animal images into multiple categories through an interactive web interface.

---

# 🚀 Features

✅ Animal Image Classification  
✅ Deep Learning Model using TensorFlow/Keras  
✅ Flask Web Application  
✅ Upload Single Image Prediction  
✅ Batch Prediction Support  
✅ Responsive UI Design  
✅ Trained MobileNetV2 Model  
✅ Real-Time Prediction Results  

---

# 🧠 Model Information

- Model: MobileNetV2
- Framework: TensorFlow / Keras
- Input Size: 224x224
- Classes: 15 Animal Categories

### Supported Classes
- Bear
- Bird
- Cat
- Cow
- Deer
- Dog
- Dolphin
- Elephant
- Giraffe
- Horse
- Kangaroo
- Lion
- Panda
- Tiger
- Zebra

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/BobbyBohra/Animal-classification-mode
```

---

## Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

#### Windows
```bash
venv\Scripts\activate
```

#### Linux / Mac
```bash
source venv/bin/activate
```

---

# 📥 Install Requirements

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Application

```bash
python app.py
```
